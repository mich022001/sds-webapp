# SDS Web App

SDS Web App is the web frontend and API layer of the SDS Direct Sales System.

## Main Features

- Member registration with sponsor hierarchy
- Registration code onboarding
- Product and package sales
- Bonus tracking
- Redemption workflow
- Reports and dashboards
- Role-based access control

## Stack

- React + Vite
- Tailwind CSS
- Supabase PostgreSQL
- Serverless API routes

## Main Folders

- `src/` → React frontend
- `api/` → backend API routes
- `src/pages/` → application pages
